# Hui Wan's Academic Homepage

A modern, responsive academic personal website built with HTML, CSS, and JavaScript.

## 🌟 Features

- **Modern Design**: Clean, professional academic style with subtle animations
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Elements**:
  - Typing effect for research interests
  - Smooth scroll animations
  - Hover effects on cards and links
- **Easy to Customize**: Well-structured code with clear CSS variables

## 🚀 Quick Start

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. Or use a local server (recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

### Deployment to GitHub Pages

1. Create a repository named `yourusername.github.io`
2. Push all files to the repository
3. Your site will be live at `https://yourusername.github.io`

## ✏️ Customization

### Update Personal Information

Edit `index.html` and update:
- Name, title, and institution
- Research interests and bio
- Publications
- Awards
- Contact information

### Change Profile Photo

Replace `images/profile.jpg` with your photo (recommended size: 400x400px).

### Modify Colors

Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #667eea;      /* Main accent color */
    --primary-dark: #764ba2;       /* Darker accent */
    --secondary-color: #4299e1;    /* Secondary color */
    --background-light: #ffffff;   /* Light background */
    --text-primary: #2d3748;       /* Primary text color */
}
```
